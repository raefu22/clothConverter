# Cloth Creator for Maya
Create cloth in Maya using Python utilizing Maya's nCloth system <br>
Make tablecloth, curtains, and ribbon bows <br>
Choose a material to make it: <br>
 - Cotton
 - Velvet
 - Satin
 - Plaid
 - Iridescent <br>
<br>
